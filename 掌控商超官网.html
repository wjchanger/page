Zhangkong Smart Retail

Bluetooth-Based Smart Retail Device System

Product Overview

Zhangkong Smart Retail is a Bluetooth-based smart retail hardware system designed for store management, device control, and automation. It is widely used in retail environments such as shelves, lighting systems, and in-store device management.

Mobile Application

The mobile application "掌控商超" is specifically developed to work with Zhangkong Smart Retail devices.

⚠️ This app requires designated hardware devices and cannot function independently.

Hardware Description

The system includes Bluetooth-enabled control modules used for retail device management. These hardware devices are developed as part of the Zhangkong Smart Retail solution.
